A complete Power BI analytics project designed to explore and visualize pizza store sales data, uncovering key business insights and performance trends.
 Project Overview
 
This project focuses on using Power BI to analyze pizza sales data and convert it into interactive, actionable business intelligence.
The dashboard highlights essential KPIs, product performance, revenue distribution, and customer ordering patterns to support data-driven decision-making
Tools & Technologies
Power BI Desktop (Dashboard, DAX, Data Modeling)
Power Query (Data Cleaning & Transformations)
DAX Measures (KPI Calculations & Dynamic Analysis)

 Key Features of the Dashboard
 Sales KPIs
Total Revenue
Total Orders
Average Order Value
Revenue Contribution by Category


 Product Analysis
Top 5 best-selling pizzas
Category-wise & size-wise revenue analysis
Profit performance by pizza type
Low-selling items for improvement
 Time-Based Insights
Daily, monthly, and yearly trends
Peak order days and peak ordering hours
Seasonal demand patterns


 Interactive Filters
Pizza category filter
Size filter
Date slicer to analyze trends over any time period


Data Preparation
All data transformations were performed using Power Query, including:
Data cleaning (removing nulls, fixing data types)
Merging data tables (orders, pizzas, order details)
Building relationships for efficient modeling
Creating calculated columns where needed


DAX Measures Used
Some examples:
Total Revenue
Total Orders
Average Order Value (AOV)
Revenue by Category
Top N Ranking Measures


Insights Generated
Identified pizzas generating maximum revenue
Found category-wise contribution to total sales
Detected peak sales timings to support staffing decisions
Highlighted slow-moving pizzas for marketing or removal
Observed variations in monthly sales trends
